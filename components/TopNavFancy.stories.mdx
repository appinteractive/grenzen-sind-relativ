# Fancy Navigation
With `MDX` we can define a story for `Checkbox` right in the middle of our
Markdown documentation.

<Canvas>
  <Story name='Fancy'>{{
    template: `
      <div class="h-full w-full bg-gray-300 absolute left-0 top-0 overflow-hidden text-gray-700 antialiased leading-tight h-64">
        <TopNavFancy :open="isOpen" :currentMain="currentMain" :activeTab="activeTab" :navigation="navigation" @change-main="setCurrentMain" @open="toggle" />
      </div>`,
    props: {
      /*isOpen: {
        default: boolean('Open', true)
      },*/
      /*currentMain: {
        default: number('Current Main', 2)
      },*/
      activeTab: {
        default: number('Active Tab', 2)
      },
      /*navigation: {
        default: object('Navigation', [])
      }*/
    },
    methods: {
      setCurrentMain(index) {
        this.currentMain = index
      },
      open(isOpen) {
        this.isOpen = isOpen
      }
    },
    data: () => ({
      isOpen: false,
      currentMain: 2,
      navigation
    })
  }}</Story>
</Canvas>
